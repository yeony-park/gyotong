# KTX 공실률 분석 데이터 및 API 정리

본 문서는 LLM 기반 자연어 보고서 생성 기능을 결합하기 전까지 수행된 데이터 수집, 전처리, 탐색 분석, 예측 모델링, 외부 변수 해석 범위를 정리한 것이다. 현재 단계의 분석 목표는 KTX 노선별 공실률 변동을 설명하고, 노선 운영자가 증편, 좌석 배분, 특정 기간 집중 관리 등 정책 판단에 참고할 수 있는 근거를 제공하는 데 있다.

|---|---|---:|---:|---|---|

공공데이터포털의 `한국철도공사_KTX 월별 역별 승하차` 데이터는 2004년 4월부터 2022년 12월까지 월별·역별 KTX 승하차 인원을 제공하는 자료이며, 공공데이터포털은 파일 데이터를 XML/JSON 오픈 API로 자동 변환해 활용할 수 있다고 안내한다. 다만 승하차 인원만으로는 공급좌석 또는 이용률을 알 수 없으므로, 해당 API만으로는 실제 공실률을 직접 산출하기 어렵다.

## 2. 전처리 및 파생 변수 생성

KTX 월별 여객·이용률 데이터는 wide 형태의 엑셀 데이터를 노선별 long 형태로 변환하였다. 이 과정에서 `KTX이용률`을 기준으로 `공실률 = 100 - KTX이용률`을 산출하고, 공실률이 0보다 작은 경우를 `초과수요`로 정의하였다. 월, 분기, 계절 변수와 함께 코로나 기간(`2020-01~2022-06`), SRT 개통 이후(`2016-12 이후`) 플래그를 생성하여 시간적·구조적 변화를 모델이 반영할 수 있도록 했다.

공실률 예측 모델에는 시간 지연 변수를 추가하였다. 사용한 lag feature는 전월 공실률(`공실률_lag1`), 전년 동월 공실률(`공실률_lag12`), 3개월 이동평균(`공실률_ma3`)이다. 이러한 변수는 현재 공실률이 직전 수요 상태와 계절적 반복성에 영향을 받을 수 있다는 가정을 반영한다.

공휴일 데이터는 한국천문연구원 특일 정보 API를 통해 2004년부터 2024년까지 수집하였다. 일별 공휴일 정보를 월 단위로 집계하여 `공휴일수`, `명절연휴포함`, `황금연휴포함`, `명절전달`, `명절후달` 등의 변수를 생성하고 KTX 월별 공실률 데이터에 결합하였다.

## 3. 파트별 분석 구성

### 파트 1. 노선·역별 현황 집계

파트 1은 예측 모델링 이전의 현황 진단 파트이다. 역별 승차·하차 인원을 기준으로 총 이용인원과 승차 비중을 계산하고, 노선-역 매핑 테이블을 결합하여 어떤 노선과 역에 수요가 집중되는지 확인했다.

현재 파트 1은 7개 노선을 다룬다.

```text
강릉선, 경부선, 경전선, 전라선, 중부내륙선, 중앙선, 호남선
```

노선별 역 수는 다음과 같다.

```text
강릉선 16개, 경부선 19개, 경전선 16개, 전라선 18개,
중부내륙선 6개, 중앙선 10개, 호남선 17개
```

역별 승차 인원 기준 상위 역은 서울, 동대구, 대전, 부산, 용산 등 주요 거점역으로 나타났다. 서울역의 승차 인원은 21,184,388명으로 전체 승차 인원의 약 14.6%를 차지하였다. 이 결과는 예측 모델의 입력이라기보다, 이후 공실률 예측 결과를 해석하기 위한 수요 규모의 기준선으로 사용된다.

### 파트 2. 시간/계절 패턴 분석

파트 2는 월별·계절별 공실률 패턴을 탐색하는 EDA 파트이다. 월, 분기, 계절별 평균 공실률을 비교하고, 최근 12개월의 공실률 추이를 확인한다. 또한 공휴일 및 명절 변수를 결합하여 명절달과 비명절달의 평균 공실률 차이를 비교하였다.

명절 효과 검정 결과, 현재 데이터에서는 모든 노선에서 p-value가 0.05보다 커서 명절 여부만으로 공실률 차이가 통계적으로 유의하다고 단정하기는 어렵다. 따라서 공휴일/명절 변수는 단독 결론보다는 외부 요인 해석을 위한 참고 신호로 활용한다.

### 파트 3-A. 공실률 회귀 예측 모델

파트 3-A는 CatBoost 회귀 모델을 사용한 메인 공실률 예측 파트이다. 목표 변수는 `공실률`이며, 주요 입력 변수는 월, 분기, 계절, 노선, 코로나기간, SRT개통후, 전월 공실률, 전년 동월 공실률, 3개월 이동평균이다. 공휴일 변수 결합 이후에는 공휴일수, 명절연휴포함, 황금연휴포함도 노선별 모델의 보조 변수로 활용되었다.

공실률 예측 모델은 5개 노선을 대상으로 학습하였다.

```text
경부선, 경전선, 동해선, 전라선, 호남선
```

5개 노선으로 제한된 이유는 현재 월별 공실률·이용률 데이터가 위 5개 노선 단위로 제공되기 때문이다. 파트 1의 7개 노선은 노선·역 현황 데이터 기준이며, 파트 3-A는 월별 공실률 시계열이 확보된 노선만 모델링 대상으로 삼았다.

통합 CatBoost 모델의 평균 MAE는 6.9959%p이다. 노선별 개별 모델의 MAE는 호남선 6.5133%p, 경부선 8.3836%p, 전라선 9.9586%p, 경전선 11.7107%p, 동해선 12.5941%p로 나타났다.

SHAP 기반 변수 중요도 평균 상위 변수는 다음과 같다.

```text
전월 공실률       5.199
3개월 이동평균    4.953
코로나기간        3.280
전년 동월 공실률  1.458
분기             1.153
공휴일수          1.036
월               0.981
계절             0.653
```

이는 공실률 예측에서 최근 공실률 흐름과 이동평균이 가장 큰 영향을 주며, 코로나 기간과 전년 동월 패턴도 중요한 설명 변수로 작용했음을 의미한다.

### 파트 3-B. 시계열 예측 모델

파트 3-B는 Prophet을 사용한 시계열 기준선 모델이다. 3-A가 여러 설명 변수를 함께 사용하는 회귀 모델이라면, 3-B는 노선별 월별 공실률의 시간 흐름, 추세, 연간 계절성만을 기준으로 예측하는 비교 모델이다.

각 노선에서 최근 12개월을 holdout test 구간으로 분리하고, 이전 기간으로 Prophet을 학습한 뒤 실제 공실률과 예측 공실률을 비교하였다. 결과는 다음과 같다.

```text
경부선 MAE 26.47%p, RMSE 26.62%p
경전선 MAE 28.51%p, RMSE 28.70%p
동해선 MAE 37.42%p, RMSE 38.86%p
전라선 MAE 19.58%p, RMSE 19.90%p
호남선 MAE 17.38%p, RMSE 17.62%p
```

Prophet은 장기 추세와 계절성을 파악하는 기준선으로는 유용하지만, 2024년처럼 실제 공실률이 음수로 내려가는 초과수요 구간은 충분히 따라가지 못했다. 따라서 3-B는 최종 의사결정 모델이라기보다, 단순 시간 패턴만으로 최근 수요 압박을 설명하기 어렵다는 점을 보여주는 보조 비교 모델로 해석한다.

### 탭 4. 외부 변수 영향 분석

대시보드의 탭 4는 원래 파트 5에 해당하는 외부 변수 영향 분석이다. 이 탭은 새로운 예측값을 만드는 화면이 아니라, 공휴일, 명절, 코로나 기간, SRT 개통 이후와 같은 외부 요인이 공실률 변화에 어떤 신호를 주는지 해석하기 위한 화면이다.

탭 4는 다음 요소로 구성된다.

```text
1. External Signal: 외부 변수의 평균 SHAP 중요도
2. Holiday Test: 명절달과 비명절달의 공실률 차이 및 p-value
3. Holiday Vacancy Gap: 노선별 명절달-비명절달 공실률 차이
4. Route External SHAP: 노선별 외부 변수 중요도
5. Policy Reading: 정책적 해석 요약
```

현재 명절 효과는 통계적으로 유의하다고 단정하기 어렵지만, 노선별 차이의 방향과 크기는 운영 정책의 참고 신호로 사용할 수 있다. 예를 들어 특정 노선에서 명절달 공실률이 낮아지거나 초과수요가 반복된다면 임시 증편, 좌석 배분 조정, 사전 예약 유도 정책의 후보로 검토할 수 있다.

## 4. 7개 노선 확장 가능성

파트 1은 노선·역 현황 데이터 기준이므로 7개 노선을 보여준다. 반면 파트 3-A와 3-B는 월별 공실률·이용률이 확보된 5개 노선만 학습 대상으로 사용한다. 공실률 예측을 7개 노선으로 확장하려면 강릉선, 중앙선, 중부내륙선에 대해서도 월별 이용률 또는 공급좌석 기반 자료가 필요하다.

공공데이터 API로 월별 역별 승하차 인원을 추가 수집하면 7개 노선의 수요 흐름은 확장할 수 있다. 그러나 승하차 인원만으로는 공실률의 분모인 공급좌석이나 운행 공급량을 알 수 없기 때문에 실제 공실률을 직접 계산할 수 없다. 7개 노선 전체로 확장하려면 다음 중 하나가 필요하다.

```text
1. 노선별 월별 공실률
2. 노선별 월별 KTX 이용률
3. 노선별 월별 수송인원 + 공급좌석 또는 편성별 좌석 수 + 운행횟수
```

따라서 현재 제출 단계에서는 “7개 노선 현황 분석 + 5개 노선 공실률 예측”으로 분석 범위를 명확히 구분하는 것이 가장 안정적이다.

## 5. 현재 단계의 한계 및 향후 LLM 보고서 연계

현재 분석은 데이터 전처리, EDA, 예측 모델, 외부 변수 해석까지 완료된 상태이며, LLM 기반 자연어 보고서 생성은 아직 연결되지 않았다. 따라서 대시보드의 인사이트 문구는 현재까지의 분석 결과를 사람이 해석하여 작성한 요약이며, LLM이 자동 생성한 보고서는 아니다.

향후 파트 6에서는 각 탭에서 산출된 핵심 지표, 예측 결과, SHAP 중요도, 외부 변수 검정 결과를 구조화된 입력으로 정리한 뒤, 노선 관리자 관점의 자연어 보고서를 생성할 수 있다. 이때 LLM 보고서는 모델 결과를 새로 계산하는 기능이 아니라, 이미 산출된 분석 결과를 정책 언어로 요약하고 설명하는 계층으로 설계하는 것이 적절하다.

## 6. 제출용 요약 문단

본 연구는 KTX 노선별 공실률의 원인과 향후 변동 가능성을 파악하기 위해 한국철도공사의 고속철도 여객·이용률 데이터, 역별 승하차 현황, KTX 구간별 통계, 공휴일 API 데이터를 결합하였다. 먼저 역별 승하차 및 노선 구성 자료를 통해 전체 7개 노선의 현황을 집계하고, 주요 거점역 및 노선별 이용 규모를 파악하였다. 이후 월별 공실률 데이터가 확보된 경부선, 호남선, 경전선, 전라선, 동해선 5개 노선을 대상으로 CatBoost 기반 공실률 회귀 예측 모델과 Prophet 기반 시계열 기준선 모델을 구축하였다.

메인 예측 모델인 CatBoost는 월, 분기, 계절, 코로나 기간, SRT 개통 이후, 전월 공실률, 전년 동월 공실률, 3개월 이동평균 등을 입력 변수로 사용하였다. SHAP 분석 결과, 전월 공실률과 3개월 이동평균이 가장 큰 영향 요인으로 나타나 최근 수요 상태의 지속성이 공실률 예측에 중요함을 확인하였다. Prophet 시계열 모델은 추세와 계절성을 기준으로 한 비교 모델로 활용하였으며, 최근 초과수요 구간은 단순 시간 패턴만으로 충분히 설명하기 어렵다는 점을 확인하였다.

외부 변수 분석에서는 공휴일 수, 명절 연휴 포함 여부, 황금연휴 여부, 코로나 기간, SRT 개통 이후와 같은 변수를 활용하였다. 명절 여부에 따른 평균 공실률 차이는 현재 데이터에서 통계적으로 강하게 유의하다고 단정하기 어렵지만, 노선별 차이 방향과 크기는 임시 증편, 좌석 배분, 특정 기간 집중 관리 등 운영 정책의 참고 신호로 활용할 수 있다. 최종적으로 본 분석은 7개 노선의 현황 진단과 5개 노선의 공실률 예측을 결합하여, 노선 관리자에게 데이터 기반 의사결정 근거를 제공하는 것을 목표로 한다.
